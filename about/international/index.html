Program Overview

Code for America is building upon its past program successes in the United States to create an international network of like-minded people and organizations who want to impact change in their respective countries. Partners from around the world are using Code for America’s resources and lessons learned to transform their own communities by improving local governments. Launched in 2013, this program is quickly gaining momentum.

For more on this program email international[at]codeforamerica.org
