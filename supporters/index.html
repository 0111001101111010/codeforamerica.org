Foundations and Individual Donors

Code for America is generously supported by leaders in philanthropy and technology. Their support helps make our work connecting city governments and web talent possible. If you are interested in giving, we would appreciate your contribution of any size. Donations to Code for America, a 501 (c)(3) nonprofit, nonpartisan organization, are tax deductible.

Code for America's EIN number is 27-1067272.
